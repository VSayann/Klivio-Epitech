## Contexte
Dans le cadre de ma formation, notre premier projet Epitech, il nous a été demandé d'intégrer une application web en réalisant une page d’accueil complète en HTML/CSS conforme à la maquette fournie totalement responsive. Deux rendus attendus, l'un uniquement avec HTML et CSS en repectant à la lettre la maquette FIgma fournie. Et l'autre en intégrant Tailwind mais avec liberté artistique totale.

### Technos utilisées :
- HTML+CSS
- Tailwind CSS
- Figma

### Exigences HTML :
- Balises sémantiques
- Titres hiérarchisés
- "*alt*" obligatoires sur les images
- Structure adaptée aux lecteurs d’écran
- Validation W3C obligatoire
- Bonnes pratiques SEO (balises meta, titres, structure)

### Exigences CSS :
- Aucun CSS inline
- Fichier CSS externe obligatoire
- Utilisation de "*Flexbox*" ou "*Grid*"
- Styles exactes (typographie, couleurs, espacements)
- Site responsive desktop + mobile
- Respect total de la maquette fournie

### Exigences Tailwind :
- Utilisation des classes utilitaires Tailwind
- ```tailwind.config.js``` personnalisable
- Liberté artistique totale
